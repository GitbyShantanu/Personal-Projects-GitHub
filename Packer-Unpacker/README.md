# Packer-Unpacker

A simple Java-based program to pack and unpack files.

## Installation
1. Ensure you have Java installed.
2. Download the project files.

## Usage
To pack files:
1. Run the `Packer` class.
2. Enter the directory containing files to be packed.
3. Enter the desired name for the packed file.

## To unpack files:
1. Run the `Unpacker` class.
2. Enter the name of the packed file to unpack.

## Known Issues
* Currently, the packed file is not encrypted.
* The header format is fixed.
